Practice MCQ Questions

1. What does CSS stand for?

  • A) Computer Style Sheets
  • B) Cascading Style Sheets
  • C) Creative Style System
  • D) Colorful Style Sheets
Correct Answer: B) Cascading Style Sheets
CSS stands for Cascading Style Sheets. It is used to style and layout web pages.

2. Which property is used to change the background color?

  • A) bg-color
  • B) background-color
  • C) color-background
  • D) background
Correct Answer: B) background-color
The background-color property is used to set the background color of an element. You can also use the shorthand 'background' property.

3. Which CSS property is used to make text bold?

  • A) text-weight: bold;
  • B) font-weight: bold;
  • C) text-style: bold;
  • D) font: bold;
Correct Answer: B) font-weight: bold;
The font-weight property is used to set the weight (thickness) of the font. Values can be normal, bold, or numeric values like 100-900.

4. What is Flexbox used for?

  • A) Creating animations
  • B) Creating flexible, responsive layouts
  • C) Adding colors to elements
  • D) Creating shadows
Correct Answer: B) Creating flexible, responsive layouts
Flexbox (Flexible Box Layout) is a CSS layout model that allows you to design flexible and responsive layouts efficiently.

5. Which property is used to add space between elements in Flexbox?

  • A) gap
  • B) space-between
  • C) margin
  • D) padding
Correct Answer: A) gap
The gap property is used to set the space between flex items. It's a shorthand for row-gap and column-gap.
